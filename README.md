-------------Library Management System---------------

A modular C++ console application that simulates a functional library environment. It manages books, users, borrowing/returning, search tools, notifications, and report generation. The system is built across 10 .cpp files and 8 .h files for clean structure and scalability.

------------Features--------------

--Add, list, borrow, and return books
--Register and manage users
--Search by title or author
--Track transactions
--Generate user activity reports
--Organized class-based architecture

---------------Project Structure---------------

--Header Files (8): Class definitions for Book, User, Library, Librarian, Catalog, Transaction, Notification, ReportGenerator
--Source Files (10): Implementations for all classes plus main program logic
